Facemask Detection Alert System

This project contains a facemask classifier model that detects whether an individual is wearing a mask or not in an image or real-time. In the case of No Mask, a warningmessage pops up, and the system sends an email to officials containing an attached image of that person. The trained model showed an accuracy of 98.33%. Python, Keras, OpenCV, SMTP, Tkinter, and CNN architecture are utilized to create this system.
